# Omarchy Hotkeys Helper

Learn Omarchy shortcuts. Track progress. Build habits.

**https://omarchy-hotkeys.vercel.app**


## Quick Start

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
bun build
bun start
```
